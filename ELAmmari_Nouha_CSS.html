Tableau des éléments CSS

Voici un tableau présentant quelques éléments essentiels de CSS issus du mémento

Catégorie Proprièté Description
Sélécteurs .class Applique un style à tous les éléments ayant une classe spécifique.
Sélécteurs #id Applique un style à un élément avec un ID unique.
Texte font-family Définit la famille de polices (ex : Arial, Times).
Texte font-weight Spécifie le poids de la police (normal, bold).
Positionnement margin Définit la marge extérieure autour d'un élément.
Positionnement padding Définit l'espace intérieur entre le contenu et la bordure.
Arriere-plan background-color Définit la couleur d'arrière-plan d'un élément.
Arriere-plan background-image Ajoute une image en arrière-plan.
Nouveautés CSS3 border-radius Crée des coins arrondis sur un élément.
Nouveautés CSS3 box-shadow Ajoute une ombre autour d'un élément.
